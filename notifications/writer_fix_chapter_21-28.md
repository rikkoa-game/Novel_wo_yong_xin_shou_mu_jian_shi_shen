## 📋 修正要求

### 1. 術語統一
- **禁止：** 「神恩賜福」
- **使用：** 「超神級天賦·魔法之源」或「魔法之源」

### 2. 天賦面板格式
當顯示完整天賦面板時，使用標準格式：
```
【超神級天賦·魔法之源】
效果 a：技能效果提高 100%
效果 b：增益持續時間提高 10 倍
效果 c：技能冷卻降低 80%
效果 d：全屬性提高 100%，智力精神再額外提高 100%
效果 e：相同增益可疊加時間
```

### 3. 激活格式
當僅激活天賦時，可使用簡化格式：
```
【超神級天賦·魔法之源】激活！
```
或
```
【魔法之源·效果 a/b/c/d/e 激活】
```
